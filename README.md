OpenRA Dune 2000 简易汉化补丁



⚠️ 说明

这是一个处于 早期测试阶段的汉化补丁。

主要汉化了单位名称、建筑名称和百科介绍。

目前仅适用于 OpenRA 的Dune 2000 (D2K)模组。



📦 包含内容

.ftl: 核心汉化文件 (单位/建筑/UI文本)。

.ttf：字体



🚀 安装步骤

1\. 汉化文本

下载本仓库中的 `.ftl` 文件，覆盖到你的游戏目录：

OpenRA目录/mods/d2k/fluent

2\. 字体替换

游戏原版字体不支持中文，必须手动替换：

1\. 找一个中文字体文件（例如电脑里的黑体 simhei.ttf），或直接使用我提供的FreeSans.ttf和FreeSansBold.ttf文件。

2\. 将其重命名为 FreeSans.ttf和FreeSansBold.ttf。

3\. 覆盖到游戏目录：OpenRA目录/mods/common。
